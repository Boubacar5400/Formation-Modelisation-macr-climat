# Formation-Modelisation-macr-climat
Formation à UPPA avec Master 2 en modelisation macroeconomie dynamique appliquee au climat , Application avec Octave et Dynare
 **Pour la reproduction des application faite il faut créer des dossiers dans les dossiers qui contients les fichers .mod : dossiers à créer : data, data_raw, figures, tables**

 Notes : **Si y'a des fichiers main.m** il faut les excécuter en premier sous octave. pas besoin de dynare directement sur le command window : on tape : *main* sans l'extension (.m) - car c'est une programme matlab

 - Pour excécuter les fichier **.mod** il faut toujours précéder par *dynare nom du fichier.mod*
